# Benchmarks

Coloque aqui a série histórica real do índice UTIL para usar como benchmark fiel à realidade.

Arquivo padrão esperado pela V3:

```text
data/benchmarks/UTIL_historico.csv
```

Formato recomendado:

```csv
date,close
2019-01-02,5723
2019-01-03,5750
```

Também são aceitas colunas como `Data`, `Fechamento`, `Pontuação`, `Pontuacao`, `Último` ou `Ultimo`.

O arquivo real não precisa ser enviado para o GitHub; pode ficar apenas no VPS.
